## 合并两个有序链表

将两个有序链表合并为一个新的有序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例：

```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```

## 解法 1：循环 1:1 比较

### 分析

创建一个新的链表，l1 和 l2 链表互相比较，小的一方或另一方没有值时进行赋值即可

### 解答

```javascript
var mergeTwoLists = function(l1, l2) {
  let res = new ListNode();
  let now = res;
  while (l1 || l2) {
    const val_1 = l1 ? l1.val : Number.MAX_SAFE_INTEGER;
    const val_2 = l2 ? l2.val : Number.MAX_SAFE_INTEGER;
    if (val_1 < val_2) {
      now.next = l1;
      l1 = l1.next;
    } else {
      now.next = l2;
      l2 = l2.next;
    }
    now = now.next;
  }
  return res.next;
};
```

## 解法 2：优化循环比较

### 分析

核心和解法 1 不变，优化的点：  
1，不创建额外的列表，备份初始值最小的链表  
2，若其中一方没有有效值时，由于两个链表都是有序链表，则另外一方直接整个赋值即可

### 解答

```javascript
function mergeTwoLists(l1, l2) {
  let root;
  if (l1 && l2) {
    if (l1.val < l2.val) {
      root = l1;
      l1 = l1.next;
    } else {
      root = l2;
      l2 = l2.next;
    }
  } else {
    return l1 || l2;
  }
  let curr = root;
  while (l1 || l2) {
    if (!l1 || !l2) {
      curr.next = l1 || l2;
      break;
    }
    if (l2.val > l1.val) {
      curr.next = l1;
      l1 = l1.next;
    } else {
      curr.next = l2;
      l2 = l2.next;
    }
    curr = curr.next;
  }
  return root;
}
```
