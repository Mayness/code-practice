## 排序链表

在 O(n log n) 时间复杂度和常数级空间复杂度下，对链表进行排序。

示例 1:

输入: 4->2->1->3
输出: 1->2->3->4
示例 2:

输入: -1->5->3->4->0
输出: -1->0->3->4->5

### 解法 1：选择排序

### 分析

循环每一位，挑选出后续最小的 1 位，将它放置在当前循环位上

### 解答

```javascript
var sortList = function(head) {
  curr = head;
  while (curr) {
    let minNode = curr;
    let node = curr;
    let flag = false;
    while (node) {
      if (node.val < minNode.val) {
        flag = true;
        minNode = node;
      }
      node = node.next;
    }
    if (flag) {
      const temp = curr.val;
      curr.val = minNode.val;
      minNode.val = temp;
    }
    curr = curr.next;
  }
  return head;
};
```
